# Kiro Configuration for OneCity.space

This directory contains Kiro-specific configuration and steering files for the OneCity.space project.

## Steering Files

Steering files provide context and guidelines to Kiro AI assistant when working on this project:

- **project-overview.md** - High-level project description and architecture
- **coding-standards.md** - TypeScript, React, and code quality guidelines
- **blockchain-integration.md** - MetaMask and smart contract interaction patterns
- **ui-styling.md** - Tailwind CSS, glassmorphism, and design system
- **testing-and-debugging.md** - Development workflow and debugging tips (manual inclusion)
- **database-integration.md** - Supabase integration patterns (auto-included for database files)
- **3d-graphics.md** - Three.js and React Three Fiber guidelines (auto-included for game components)

## Inclusion Types

- **always** - Included in all Kiro interactions
- **fileMatch** - Included when working with matching files
- **manual** - Included only when explicitly referenced with `#` in chat

## Usage

These files help Kiro understand:
- Project structure and conventions
- Technology stack and patterns
- Best practices specific to this codebase
- Common issues and solutions

When working with Kiro, reference these guidelines or let them automatically provide context based on the files you're editing.
